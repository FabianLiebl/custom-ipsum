Custom Ipsum

by FallenSquirrel

You currently have Ipsum Records saved

Activate fill action by using Right click -> Fill with Custom Ipsum or by ALT + clicking on an input field.

When activating the fill action, Custom Ipsum will randomly choose one of the appropriate type of saved Ipsum Records for the input field.

Supports fields of type Textarea, Input[type=text], Input[type=email], Input[type=number] as well as some (but not all) Richtext editor types.

If the options Guess Email fields or Guess Url fields are active, might decide to fill Input[type=text] fields with email-addresses or link-urls based on the fields' label/placeholder.
